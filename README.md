# Lost and Found App 7.1P

This is a simple Android Lost and Found app created for SIT305 Task 7.1P.

## App Overview

The app allows users to create lost or found item adverts. Each advert can include item details, contact information, location, category, image, and a posted date/time stamp.

## Features

- Create a lost or found advert
- Save advert details using a Room database
- View all lost and found adverts in a list
- Tap an advert to view full details
- Remove an advert once the item has been returned or resolved
- Filter adverts by category
- Upload an image for each advert
- Automatically save a date/time stamp when the advert is posted

## Technologies Used

- Java
- Android Studio
- Room Database
- XML layouts
- GitHub

## How to Use the App

1. Open the app.
2. Select **Create a New Advert**.
3. Choose whether the post is **Lost** or **Found**.
4. Enter the item details.
5. Add a category such as Electronics, Pets, or Wallets.
6. Choose an image.
7. Press **Save**.
8. Go to **Show All Lost and Found Items** to view saved adverts.
9. Use the category filter to search for specific types of items.
10. Tap an advert to view details or remove it.
